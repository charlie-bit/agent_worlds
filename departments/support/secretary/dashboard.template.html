Getting Started

  1. 1
    Choose your model

    Open Config → select Model / Host (default: Claude Code). This tells the secretary which adapter to use when collecting data.

  2. 2
    Enable sources

    Toggle on the sources you want (Slack is first). For Slack, pick which channels to monitor from the list.

  3. 3
    Save config

    Click ↓ Download registry.yaml and move the file to ~/.agent_worlds/secretary/registry.yaml.

  4. 4
    Run the secretary

    Ask Claude: "Run the daily digest for today." It will collect, triage, and write outbox/YYYY-MM-DD-digest.md.

  5. 5
    Refresh dashboard

    Run python3 generate_dashboard.py to render the new digest into this dashboard.

Changes to Config auto-download — no manual save needed.
